# Lara & Sons Differentials & 4x4

Professional website for **Lara & Sons Differentials & 4x4** — a family-owned differential repair, rebuild, and custom 4x4 shop based in Jurupa Valley, CA.

## Tech Stack

- **Vite** — Fast development server and build tool
- **Vanilla HTML/CSS/JS** — Lightweight, no heavy frameworks
- **Font Awesome** — Icon library
- **SVG** — Custom circular logo with arched text

## Development

```bash
npm install
npm run dev    # Start dev server
npm run build  # Build for production
```

## Features

- Single-page responsive design
- Services, Parts, About, Gallery, and Contact sections
- SEO-optimized with meta tags and Open Graph
- Mobile-first responsive layout
- Custom confetti effects
- Gallery lightbox
- Contact inquiry form
